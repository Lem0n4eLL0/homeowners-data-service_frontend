# homeowners-data-service frontend

> Frontend-приложение на стеке **React 19 + Vite + TypeScript**

[![Vite](https://img.shields.io/badge/Vite-7.3.1-646CFF?logo=vite)](https://vitejs.dev)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.1.6-3178C6?logo=typescript)](https://www.typescriptlang.org)
[![Storybook](https://img.shields.io/badge/Storybook-10.2.8-FF4785?logo=storybook)](https://storybook.js.org)

## Стек технологий

- **Vite** — быстрый сборщик и dev-сервер
- **React 19** — библиотека для построения UI
- **TypeScript** — статическая типизация
- **Storybook** — изолированная разработка компонентов
- **Jest + Vitest + Playwright** — многоуровневое тестирование
- **ESLint + Stylelint + Prettier** — контроль качества кода
- **Husky** — git-хуки для пре-коммит проверок

## Быстрый старт

### Требования

- Node.js >= 18
- npm / yarn / pnpm

### Установка зависимостей

```bash
npm install
```

### Запуск проекта

```bash
# Dev-сервер с HMR
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшен-сборки
npm run preview
```

## Тестирование

# Unit-тесты (Jest)

```bash
npm run test # Запуск тестов
```
